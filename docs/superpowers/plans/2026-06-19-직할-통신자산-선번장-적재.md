# 강원 직할 통신자산·선번장 실데이터 적재 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** `직할/` 엑셀 6종에서 강원 직할 13개 국소의 통신자산·OFD 선번장·OPGW 백본을 추출해 현재 Cable 기반 모델로 적재하고, 가짜 시드를 교체한다(레이어 ①물리광백본+②설비).

**Architecture:** 3단계 — (1) python 오프라인 추출 → **검수된 JSON 데이터파일**(`backend/prisma/seed/data/jikhal/`), (2) TS 시드 로더가 JSON을 읽어 결정적 UUID로 Substation/Floor/Asset/Cable 생성, (3) `seed.ts`에서 `seedGangwonSubstations`→`seedJikhalAssets` 교체. 추출(지저분)과 시드(결정적)를 JSON 계약으로 분리.

**Tech Stack:** python3+openpyxl(추출), Node/Prisma/TS(시드), Vitest(시드 단위/통합 테스트), python unittest(정규화 헬퍼).

## Global Constraints

- DB: dev compose만 사용(이미 떠 있음). **docker build/up 금지.** psql: `docker exec ict-twin-postgres psql -U postgres -d ict_digital_twin -c "<sql>"`.
- 백 빌드/테스트: `cd backend && npx tsc --noEmit && npx vitest run`. 단일: `npx vitest run <path>`.
- 시드는 **컨테이너 시작마다 실행**됨(Dockerfile CMD). 초기 조직/자산 시드는 **첫 배포(빈 DB)에만** 실행되도록 `seed.ts`의 첫-배포 가드 안에 둔다(기존 패턴, 운영데이터 보존).
- **스키마 컬럼 추가/마이그레이션 0건.** 신규 필드는 `Asset.attributes`/`Cable.specParams`(JSON)로만.
- 편집/표시는 기존 편집기 범위만(자산 스칼라 8필드 + 선번장 4필드). 부가 메타는 시드 저장 전용(스펙 §4.1).
- 결정적 id: `juuid(typeNum, key)` = `9a0001${TT}-0000-4000-b000-${sha1("TT:key")[:12]}` (floor plan API의 uuid 검증 통과 형식; version=4·variant=b).
- 기존 자산종류 재사용: OFD, OFD-SLOT, RACK, OPT-COT(통합단말), OPT-SMALL(소형광), EQP-OPT-TERM(송변전광단말), PCM, PITR, EQP-PITR-2000, EQP-PITR-5000, RTU, EQP-RTU, EQP-SCADA, EQP-NET-SW.
- 광 CableCategory 재사용: `CBL-OPT`(광케이블=OPGW 백본), `CBL-OPJ`(광점퍼코드=OUT 코어). **신규 카테고리 만들지 않는다.**
- 13 직할 국소: (구)춘천,(신)춘천,북춘천,남춘천,서홍천,열병합,인제,양구,철원,화천,화천HP,춘천HP,소양HP.

- [ ] **Step 3: 구현** `scripts/extract-jikhal/normalize.py`:
```python
import re

# 13 직할 key → 표준 표시명(substations.json name 과 일치)
DIRECT = {
    "guchuncheon":"(구)춘천S/S","sinchuncheon":"(신)춘천S/S","bukchuncheon":"북춘천S/S",
    "namchuncheon":"남춘천S/S","seohongcheon":"서홍천S/S","yeolbyeonghap":"열병합S/Y",
    "inje":"인제S/S","yanggu":"양구S/S","cheorwon":"철원S/S","hwacheon":"화천S/S",
    "hwacheonhp":"화천H/P","chuncheonhp":"춘천H/P","soyanghp":"소양강H/P",
}
# 원문 별칭(정규화 후 문자열) → key. (구)/(신)춘천 모호는 검수에서 별도 처리.
ALIAS = {
    "구춘천":"guchuncheon","신춘천":"sinchuncheon","북춘천":"bukchuncheon","남춘천":"namchuncheon",
    "서홍천":"seohongcheon","열병합":"yeolbyeonghap","춘천열병합":"yeolbyeonghap",
    "인제":"inje","양구":"yanggu","철원":"cheorwon","화천":"hwacheon",
    "화천hp":"hwacheonhp","춘천hp":"chuncheonhp","소양hp":"soyanghp","소양강":"soyanghp",
}

def _clean(raw: str) -> str:
    s = (raw or "").lower()
    s = re.sub(r"\s+", "", s)
    s = s.replace("s/s","").replace("s/y","").replace("h/p","hp").replace("s/t","")
    s = re.sub(r"#\d+", "", s)
    s = re.sub(r"\(.*?\)", "", s)  # (구)/(신)/(좌측N열) 등 — (구)(신)춘천은 검수에서 보정
    s = s.replace("비금속","").replace("경로a","").replace("경로b","")
    return s

def normalize_station(raw):
    if not raw: return None
    c = _clean(raw)
    if not c: return None
    if c in ALIAS: return ALIAS[c]
    if c in DIRECT: return c
    # 외부 국소: 표준화한 원문 그대로 external key 로(접두사 ext:)
    return "ext:" + c

def is_direct(key): return key in DIRECT
def pair_key(a, b): return tuple(sorted([a, b]))
```

```ts
import { createHash } from 'crypto';
import { PrismaClient } from '@prisma/client';
import { readFileSync } from 'fs';
import { join } from 'path';

interface SubRow { key: string; name: string; isExternal: boolean; }
interface AssetRow { key: string; subKey: string; typeCode: string; name: string; parentKey: string | null; attributes?: Record<string, unknown>; }
interface CableRow { key: string; kind: 'OPGW' | 'CORE'; sourceKey: string; targetKey: string; sourceRole: string | null; targetRole: string | null; number: number | null; categoryCode: string; specParams: Record<string, unknown>; }

/** 결정적 UUID — 9a0001TT-0000-4000-b000-<sha1("TT:key")[:12]>. floor plan uuid 검증 통과. */
export function juuid(typeNum: number, key: string): string {
  const tt = String(typeNum).padStart(2, '0');
  const h = createHash('sha1').update(`${tt}:${key}`).digest('hex').slice(0, 12);
  return `9a0001${tt}-0000-4000-b000-${h}`;
}
const T = { sub: 1, floor: 2, asset: 3, cable: 4 } as const;

/** 부모(parentKey=null) 먼저 — 2레벨(OFD→slot, rack→module) 정렬이면 충분. */
export function orderAssets<A extends { parentKey: string | null }>(rows: A[]): A[] {
  return [...rows].sort((a, b) => (a.parentKey ? 1 : 0) - (b.parentKey ? 1 : 0));
}

function load<T>(name: string): T {
  return JSON.parse(readFileSync(join(__dirname, 'data', 'jikhal', name), 'utf8')) as T;
}

export async function seedJikhalAssets(prisma: PrismaClient, adminId: string, branchId: string): Promise<void> {
  const subs = load<SubRow[]>('substations.json');
  const assets = load<AssetRow[]>('assets.json');
  const cables = load<CableRow[]>('fiberCables.json');

  const typeId = new Map((await prisma.assetType.findMany({ select: { id: true, code: true } })).map((t) => [t.code, t.id]));
  const catId = new Map((await prisma.cableCategory.findMany({ select: { id: true, code: true } })).map((c) => [c.code, c.id]));
  const isExternal = new Map(subs.map((s) => [s.key, s.isExternal]));

  for (let i = 0; i < subs.length; i++) {
    const s = subs[i];
    await prisma.substation.create({ data: { id: juuid(T.sub, s.key), branchId, name: s.name, sortOrder: i, createdById: adminId, updatedById: adminId } });
    if (!s.isExternal) {
      await prisma.floor.create({ data: { id: juuid(T.floor, s.key), substationId: juuid(T.sub, s.key), name: '통신실', floorNumber: '1F', sortOrder: 0, createdById: adminId, updatedById: adminId } });
    }
  }
  for (const a of orderAssets(assets)) {
    const tId = typeId.get(a.typeCode);
    if (!tId) throw new Error(`AssetType 없음: ${a.typeCode} (${a.key})`);
    await prisma.asset.create({ data: {
      id: juuid(T.asset, a.key), substationId: juuid(T.sub, a.subKey), assetTypeId: tId,
      parentAssetId: a.parentKey ? juuid(T.asset, a.parentKey) : null,
      floorId: isExternal.get(a.subKey) ? null : juuid(T.floor, a.subKey),
      name: a.name, attributes: a.attributes ?? {}, sortOrder: 0,
    } });
  }
  for (const c of cables) {
    const cat = catId.get(c.categoryCode) ?? null;
    await prisma.cable.create({ data: {
      id: juuid(T.cable, c.key), sourceAssetId: juuid(T.asset, c.sourceKey), targetAssetId: juuid(T.asset, c.targetKey),
      cableType: 'FIBER', categoryId: cat, sourceRole: c.sourceRole, targetRole: c.targetRole, number: c.number,
      specParams: c.specParams, substationId: c.kind === 'OPGW' ? null : juuid(T.sub, sourceSub(assets, c.sourceKey)),
    } });
  }
}

/** sourceKey(asset) 의 subKey. */
function sourceSub(assets: { key: string; subKey: string }[], key: string): string {
  const a = assets.find((x) => x.key === key);
  if (!a) throw new Error(`asset 없음: ${key}`);
  return a.subKey;
}
```
