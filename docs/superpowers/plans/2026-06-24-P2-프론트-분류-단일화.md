> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans. Steps use checkbox (`- [ ]`).

**Goal:** 자산 분류를 `assetType.role`(AssetRole 유니온) **직접 비교 한 가지**로, 케이블을 `displayGroup` 한 가지로 통일한다. `isOfd`/`EquipmentKind`/`kindOf`/`useKindToAssetTypeId`/`MATERIAL_TO_CABLE_TYPE`/폴백 등 **흩어진 헬퍼·중복·데드코드를 삭제**하여 코드를 줄인다. role 은 P1 backfill + 모든 DTO/생성 지점에서 항상 채워지므로 **폴백 없음**.

**Architecture:** (1) role 을 모든 자산 출처에 채운다(백엔드 DTO + staged/파생 생성). (2) 분류 readers 를 전부 `x.role === '<role>'` 로 바꾸고 분류 헬퍼들을 삭제. (3) 에디터 `EquipmentKind` 시스템을 데이터 기반 insert bar(`usePlaceableTypes`)로 대체하고 enum/kindOf/mapper 삭제. (4) 케이블 cableType 은 `displayGroup` 파생, 자재-code 맵 삭제. (5) 프론트 `AssetType` 타입에서 미사용 필드 제거.

**Tech Stack:** React 18 + TS, Zustand, React Query, vitest; Express/Prisma(DTO/mapper). 검증: 프론트 `npx vitest run`+`npm run build`, 백엔드 `npx vitest run`, **데드코드 grep = 0**.

```ts
import { useMemo } from 'react';
import { useAssetTypes } from '../assets/hooks/useAssetTypes';
import type { AssetRole } from '../../types/asset';
export interface PlaceableType { id: string; name: string; role: AssetRole }
const FLOOR: AssetRole[] = ['rack','ofd','panel','standalone'];
const ORDER: Record<string, number> = { rack:0, ofd:1, panel:2, standalone:3 };
export function usePlaceableTypes(): PlaceableType[] {
  const { data: types = [] } = useAssetTypes();
  return useMemo(() => types.filter((t) => FLOOR.includes(t.role))
    .map((t) => ({ id: t.id, name: t.name, role: t.role }))
    .sort((a, b) => (ORDER[a.role]-ORDER[b.role]) || a.name.localeCompare(b.name)), [types]);
}
```
