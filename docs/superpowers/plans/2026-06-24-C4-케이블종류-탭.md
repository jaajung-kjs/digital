> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans. Steps use checkbox (`- [ ]`).

**Goal:** 자산관리 화면의 **케이블종류 탭**을 만든다 — 사용자가 **그룹(이름+색)** + 그 안의 **이름들**을 카탈로그 워킹카피로 CRUD(원자적 저장). 그리고 에디터 케이블 insert bar/CableSpecModal 를 **사용자 그룹 기반**으로(고정 5종 제거, C3 이월 Task 5).

**Architecture:** P4 의 `catalogStore`(범용 overlay 엔진)에 `cableGroups`/`cableCategories` 컬렉션을 **추가**(별도 스토어 아님), 백엔드 `commitCatalog` 트랜잭션에 두 컬렉션 델타를 더한다. 케이블종류 탭 UI 는 설비종류 탭과 동형 + 그룹 색. insert bar 는 `useCableGroups`(사용자 그룹).

**Tech Stack:** React 18 + TS, Zustand, React Query, Express/Prisma($transaction), vitest.
