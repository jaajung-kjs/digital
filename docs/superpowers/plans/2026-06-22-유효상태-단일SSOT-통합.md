# 유효상태 단일 SSOT 통합 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans. Steps use checkbox (`- [ ]`).

**Goal:** 자산·케이블 클라이언트 소스 4개(slim 피드 / 워킹카피 saved / overlays / nodeAssets)를 워킹카피 단일 SSOT로 통합 — `effective = saved ∪ overlays − deletes`를 유일 읽기 API로, 피드는 hydration 입력으로, 직접 피드 읽기를 ESLint로 차단. 저장/미저장 갭 구조적 제거 + 죽은 헬퍼 제거.

**Architecture:** 서버 피드(`/assets`,`/cables`,`/nodes/:id/assets`,`/workingcopy`)는 모두 워킹카피 `saved`를 채우는 hydration 입력. slim 행은 "detail 필드 null인 완전한 Asset"으로 적재돼 소비처가 부분객체에 안 깨짐. 방문/노드열람 시 detail로 승급. 모든 뷰는 `useEffectiveAssets()`/`useEffectiveCables()`(전역) 또는 그로 만든 `useTraceGraph` 그래프만 읽음.

**Tech Stack:** React 18, Zustand(+zundo), React Query, TypeScript, Vitest.

## Global Constraints

- 서버 API 변경 없음(`/assets`,`/cables`,`/nodes/:id/assets`,`/workingcopy` 그대로).
- 워킹카피 commit/OCC/undo(zundo temporal) 로직 변경 없음 — hydration은 `temporal.pause()` 안에서(비기록).
- 각 태스크 종료 시 게이트: `npx tsc --noEmit`(0), `npx vitest run`(그린), 마지막 태스크에서 `npm run build`.
- 개발은 DB만 docker + `npm run dev`. Docker 빌드 금지.
- lite 행 = 완전한 Asset 형태(detail 필드 null). 부분객체 금지.

```ts
// slimToAsset.ts
import type { Asset } from '../../types/asset';
import type { SlimAssetDTO, TraceCableInput } from '../trace/traceGraph';
import type { Cable } from '../../types/cable';

/** slim 전역 피드 행 → detail 필드 null 인 완전한 Asset(미방문 변전소용 경량 행). 방문 시 /workingcopy 가 승급. */
export function slimToAsset(s: SlimAssetDTO): Asset {
  return {
    id: s.id, substationId: s.substationId, assetTypeId: '',
    assetType: { id: '', code: s.code ?? '', name: '', group: null, displayColor: null,
      fieldTemplate: null, placementKind: null, connectionKind: s.connectionKind },
    name: s.name, parentAssetId: s.parentAssetId, floorId: null, roomText: null,
    sourcePresetId: null, installDate: null, warrantyUntil: null, replaceDue: null,
    manager: null, description: null, status: null, sortOrder: 0, updatedAt: '',
    positionX: null, positionY: null, width2d: null, height2d: null, rotation: null,
    totalU: null, slotIndex: s.slotIndex, slotSpan: null,
  } as Asset;
}

/** slim 전역 케이블 → 완전한 Cable(미방문 행). */
export function slimCableToCable(c: TraceCableInput): Cable {
  return {
    id: c.id, cableType: c.cableType ?? 'FIBER',
    sourceAssetId: c.sourceAssetId ?? null, targetAssetId: c.targetAssetId ?? null,
    sourceRole: (c.sourceRole ?? null) as Cable['sourceRole'], targetRole: (c.targetRole ?? null) as Cable['targetRole'],
    number: c.number ?? null, specParams: c.specParams ?? null,
    categoryId: c.categoryId ?? null, categoryName: c.categoryName ?? null, displayColor: c.displayColor ?? null,
    substationId: null, specification: null, pathPoints: null, updatedAt: '',
  } as Cable;
}
```
(실제 Asset/Cable 필드는 `src/types/asset.ts`,`src/types/cable.ts` 에 맞춰 채움 — 누락 필드는 null/기본값. tsc 가 강제.)
