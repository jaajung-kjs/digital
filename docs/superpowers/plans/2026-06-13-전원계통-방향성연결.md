# 전원계통 + 방향성 연결 모델 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 전원계통을 SSOT 위 "계통" 그리드 뷰로 넣고, 그 토대로 단일 방향성 연결 모델(`AssetType.connectionKind` + 케이블 IN/OUT 역할 + 방향성 트레이서)을 도입해 `root`·`BRANCH` 예외를 제거한다.

**Architecture:** 모든 것은 자산(+parentAssetId 중첩) + 케이블(흐름). CB = "피더→부하" 출력 케이블(차단기 데이터는 케이블 specParams, 필드는 전원 CableCategory.specTemplate 정의). 1대다(distributor) 자산은 케이블 IN/OUT으로 방향을 갖고, 트레이서가 IN을 따라 발전원까지 상류 추적(→ root 폐기). 계통 그리드 = 분전반/피더의 케이블-속성 레지스트리(선번장과 동형).

**Tech Stack:** Prisma/PostgreSQL, Node/Zod(backend), React/TS/Zustand(frontend), Vitest + @testing-library/react.

**참조 스펙:** `docs/superpowers/specs/2026-06-13-전원계통-방향성연결모델-design.md`

- [ ] **Step 3: 구현** — Create `frontend/src/features/power/powerTrace.ts`:
```ts
interface TraceAsset { id: string; connectionKind?: string | null }
interface TraceCable {
  id: string;
  sourceAssetId?: string | null;
  targetAssetId?: string | null;
  sourceRole?: 'IN' | 'OUT' | null;
  targetRole?: 'IN' | 'OUT' | null;
}
export interface PowerTraceNode { assetId: string; cableId: string | null }

const MAX_DEPTH = 50;

/**
 * 전원 상류 추적 — distributor 자산을 IN 케이블 따라 거슬러 올라간다.
 *  - 어떤 자산에서: 그 자산이 끝점인 케이블 중, *상류로 가는* 것을 고른다.
 *    · 자산이 distributor 면: 그 자산 쪽 역할이 'IN' 인 케이블(= 자산으로 들어오는 공급).
 *    · 아니면(부하 등): 닿은 케이블 아무거나(보통 1개) 상대편으로.
 *  - 다음 자산이 또 distributor 면 반복. 입력 케이블 없는 distributor(=발전원)에서 종료.
 *  방문 자산 set 으로 사이클 차단.
 */
export function tracePowerUpstream(
  startAssetId: string,
  cables: TraceCable[],
  assets: TraceAsset[],
): PowerTraceNode[] {
  const kindOf = new Map(assets.map((a) => [a.id, a.connectionKind ?? null]));
  const path: PowerTraceNode[] = [{ assetId: startAssetId, cableId: null }];
  const visited = new Set<string>([startAssetId]);
  let current = startAssetId;
  let depth = 0;

  while (depth++ < MAX_DEPTH) {
    const isDist = kindOf.get(current) === 'distributor';
    let next: { assetId: string; cableId: string } | null = null;
    for (const c of cables) {
      const isSource = c.sourceAssetId === current;
      const isTarget = c.targetAssetId === current;
      if (!isSource && !isTarget) continue;
      // distributor 면 '입력(IN)' 케이블만 상류. 비-distributor 면 닿은 케이블 따라감.
      if (isDist) {
        const myRole = isSource ? c.sourceRole : c.targetRole;
        if (myRole !== 'IN') continue;
      }
      const other = isSource ? c.targetAssetId : c.sourceAssetId;
      if (!other || visited.has(other)) continue;
      next = { assetId: other, cableId: c.id };
      break;
    }
    if (!next) break; // 발전원(입력 없음) 또는 끝
    visited.add(next.assetId);
    path.push(next);
    current = next.assetId;
  }
  return path;
}
```
