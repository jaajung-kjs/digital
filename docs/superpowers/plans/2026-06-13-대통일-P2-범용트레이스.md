# 연결 모델 대통일 — P2 범용 cableTrace Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 모든 연결 추적을 대체할 **단일 순수 함수 `cableTrace`** 를 작성한다 — 같은 cableType 연결을 순회하되 노드 connectionKind(none/distributor/conduit)로 *전이 규칙*만 분기. 라이브 뷰·레거시 trace는 **건드리지 않음**(병행). 합성 데이터 단위테스트로만 검증(실데이터는 P4 이후).

**Architecture:** `cableTrace(start, cableType, assets, cables)` → 방문한 노드/엣지(연결된 물리 경로). 전이: passive=들어온 것 빼고 전부(분기), distributor=OUT↔IN 수렴(형제 OUT로 안 샘), conduit=OUT#K→IN(OPGW, 채널 K 운반)→대국 OUT#K(같은 번호). 방문 엣지 집합으로 사이클(링) 안전. **이 함수가 광경로·전원상류·평면경로·토폴로지를 전부 산출**(뷰는 투영 — P3/P4).

**Tech Stack:** TypeScript, Vitest. (프론트 `frontend/src/features/trace/` 신규.)

**참조 스펙:** `docs/superpowers/specs/2026-06-13-연결모델-대통일-design.md` §2.3.

### 계약
```ts
export interface TraceAsset { id: string; connectionKind?: 'distributor' | 'conduit' | null }
export interface TraceCable {
  id: string;
  cableType?: string | null;
  sourceAssetId?: string | null;
  targetAssetId?: string | null;
  sourceRole?: 'IN' | 'OUT' | null;   // distributor/conduit 끝의 역할
  targetRole?: 'IN' | 'OUT' | null;
  number?: number | null;             // 슬롯/포트/CB 번호 (conduit 채널)
}
/** 방문한 노드·엣지(연결된 물리 경로). 뷰가 투영. */
export interface TraceResult { nodeIds: string[]; cableIds: string[] }
export function cableTrace(
  startAssetId: string,
  cableType: string,
  assets: TraceAsset[],
  cables: TraceCable[],
): TraceResult
```

### 전이 규칙 (노드에서 다음 엣지 고르기)
한 자산 A 에 케이블 `via`(없으면 시작)로 도착, 채널 `ch`(conduit OPGW 운반 번호, 없으면 null):
- A 의 cableType 케이블들 중 `via` 제외 = `others`.
- `roleAtA(c)` = c.sourceAssetId===A ? sourceRole : targetRole. `numberOf(c)` = c.number.
- **none/passive**: `others` 전부(분기).
- **distributor**: 도착 역할이 'OUT'이면 → A의 IN 케이블들; 'IN'이면 → A의 OUT 케이블들; 시작(via 없음)이면 → `others` 전부. (형제 OUT↔OUT 직접 이동 금지로 fan-out 안 샘.)
- **conduit**: 도착 역할 'OUT'이면 → A의 IN 케이블들, **채널 = via.number 운반**; 도착 역할 'IN'이면(=OPGW로 ch 들고 옴) → A의 OUT 케이블 중 **number===ch** 인 것, 채널 해제; 시작이면 → `others` 전부.

### Step 3: 구현 — Create `frontend/src/features/trace/cableTrace.ts`:
```ts
export interface TraceAsset { id: string; connectionKind?: 'distributor' | 'conduit' | null }
export interface TraceCable {
  id: string;
  cableType?: string | null;
  sourceAssetId?: string | null;
  targetAssetId?: string | null;
  sourceRole?: 'IN' | 'OUT' | null;
  targetRole?: 'IN' | 'OUT' | null;
  number?: number | null;
}
export interface TraceResult { nodeIds: string[]; cableIds: string[] }

const MAX = 5000; // 폭주 가드

/**
 * 범용 케이블 추적 — 같은 cableType 으로 물리 연결된 경로(노드/엣지).
 * 노드 connectionKind 로 전이만 분기: passive=전부, distributor=OUT↔IN 수렴,
 * conduit=OUT#K→IN(OPGW, 채널 K)→대국 OUT#K. 방문 엣지로 사이클 안전.
 */
export function cableTrace(
  startAssetId: string,
  cableType: string,
  assets: TraceAsset[],
  cables: TraceCable[],
): TraceResult {
  const kindOf = new Map(assets.map((a) => [a.id, a.connectionKind ?? null]));
  const typed = cables.filter((c) => c.cableType === cableType);
  const at = new Map<string, TraceCable[]>(); // assetId → 닿는 케이블들
  for (const c of typed) {
    if (c.sourceAssetId) (at.get(c.sourceAssetId) ?? at.set(c.sourceAssetId, []).get(c.sourceAssetId)!).push(c);
    if (c.targetAssetId) (at.get(c.targetAssetId) ?? at.set(c.targetAssetId, []).get(c.targetAssetId)!).push(c);
  }
  const roleAt = (c: TraceCable, a: string) => (c.sourceAssetId === a ? c.sourceRole : c.targetRole) ?? null;
  const other = (c: TraceCable, a: string) => (c.sourceAssetId === a ? c.targetAssetId : c.sourceAssetId) ?? null;

  const nodeIds = new Set<string>([startAssetId]);
  const cableIds = new Set<string>();
  const seen = new Set<string>(); // "cableId|towardAssetId" 전이 방문(사이클 안전)
  // 스택: 도착 자산, 도착 케이블(없으면 null), 운반 채널(conduit)
  const stack: { a: string; via: TraceCable | null; ch: number | null }[] = [
    { a: startAssetId, via: null, ch: null },
  ];
  let guard = 0;

  while (stack.length && guard++ < MAX) {
    const { a, via, ch } = stack.pop()!;
    const kind = kindOf.get(a) ?? null;
    const here = (at.get(a) ?? []).filter((c) => c.id !== via?.id);
    const entryRole = via ? roleAt(via, a) : null;

    let exits: TraceCable[];
    let nextCh: (c: TraceCable) => number | null = () => null;
    if (kind === 'distributor') {
      if (entryRole === 'OUT') exits = here.filter((c) => roleAt(c, a) === 'IN');
      else if (entryRole === 'IN') exits = here.filter((c) => roleAt(c, a) === 'OUT');
      else exits = here; // 시작
    } else if (kind === 'conduit') {
      if (entryRole === 'OUT') { exits = here.filter((c) => roleAt(c, a) === 'IN'); nextCh = () => via!.number ?? null; }
      else if (entryRole === 'IN') exits = here.filter((c) => roleAt(c, a) === 'OUT' && (c.number ?? null) === ch);
      else exits = here; // 시작
    } else {
      exits = here; // passive
    }

    for (const c of exits) {
      const nb = other(c, a);
      if (!nb) continue;
      const key = `${c.id}|${nb}`;
      if (seen.has(key)) continue;
      seen.add(key);
      cableIds.add(c.id);
      nodeIds.add(nb);
      stack.push({ a: nb, via: c, ch: nextCh(c) });
    }
  }
  return { nodeIds: [...nodeIds], cableIds: [...cableIds] };
}
```

describe('cableTrace — 엣지케이스', () => {
  it('passive 분기: 허브에서 여러 갈래 모두 포함', () => {
    // H(passive) ─c1─ A, H ─c2─ B, H ─c3─ C

    const cables = [
      { id: 'c1', cableType: 'LAN', sourceAssetId: 'H', targetAssetId: 'A' },
      { id: 'c2', cableType: 'LAN', sourceAssetId: 'H', targetAssetId: 'B' },

    const r = cableTrace('A', 'LAN', assets, cables);
    expect(new Set(r.nodeIds)).toEqual(new Set(['A', 'H', 'B', 'C']));
  });

  it('링(사이클)에서 무한루프 없이 전부 1회 방문', () => {

    const cables = [
      { id: 'a', cableType: 'FIBER', sourceAssetId: 'N1', targetAssetId: 'N2' },
      { id: 'b', cableType: 'FIBER', sourceAssetId: 'N2', targetAssetId: 'N3' },
      { id: 'c', cableType: 'FIBER', sourceAssetId: 'N3', targetAssetId: 'N1' },

    const r = cableTrace('N1', 'FIBER', assets, cables);
    expect(new Set(r.nodeIds)).toEqual(new Set(['N1', 'N2', 'N3']));
    expect(new Set(r.cableIds)).toEqual(new Set(['a', 'b', 'c']));
  });

  it('cableType 다르면 안 탐(전원 추적이 광/네트워크로 안 샘)', () => {
    const assets = [{ id: 'X', connectionKind: null }, { id: 'Y', connectionKind: null }];
    const cables = [
      { id: 'ac', cableType: 'AC', sourceAssetId: 'X', targetAssetId: 'Y' },
      { id: 'lan', cableType: 'LAN', sourceAssetId: 'X', targetAssetId: 'Y' },
    ];
    const r = cableTrace('X', 'AC', assets, cables);
    expect(new Set(r.cableIds)).toEqual(new Set(['ac']));

      { id: 'c1', cableType: 'DC', sourceAssetId: 'F', targetAssetId: 'L', sourceRole: 'OUT' as const, targetRole: null },
      { id: 'c2', cableType: 'DC', sourceAssetId: 'C', targetAssetId: 'F', sourceRole: 'OUT' as const, targetRole: 'IN' as const },
      { id: 'c3', cableType: 'DC', sourceAssetId: 'AC', targetAssetId: 'C', sourceRole: null, targetRole: 'IN' as const },

    expect(new Set(r.nodeIds)).toEqual(new Set(['L', 'F', 'C', 'AC']));
