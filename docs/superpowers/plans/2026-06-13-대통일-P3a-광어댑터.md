# 연결 모델 대통일 — P3a 광→통일 그래프 어댑터 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development. Steps use checkbox (`- [ ]`).

**Goal:** 기존 광 데이터(FiberPath + fiber 케이블 + FiberCore)를 *읽을 때* 통일 그래프(conduit 슬롯 가상자산 + OPGW 가상케이블 + 번호 OUT 가상케이블)로 변환하는 **순수 어댑터** `fiberToUnifiedGraph` 를 만든다. 이걸로 `cableTrace`가 **실데이터에서 올바른 광경로를 내는지 증명**(파괴적 마이그레이션 전, 되돌리기 가능). 라이브 뷰·데이터·레거시 **무변경**.

**Architecture:** `fiberToUnifiedGraph(fiberPaths, fiberCables, fiberCores)` → `{ assets: TraceAsset[], cables: TraceCable[] }`. FiberPath 하나당 슬롯 2개(양 OFD-end, conduit) + OPGW 케이블(IN-IN) + fiber 케이블마다 번호 OUT 케이블(설비↔슬롯). 그 그래프에 `cableTrace` 돌려 설비→슬롯→OPGW→대국슬롯→설비 경로 검증. 순수 함수 + 단위테스트만.

**Tech Stack:** TypeScript, Vitest. `frontend/src/features/trace/`.

**참조:** 스펙 `...연결모델-대통일-design.md` §2.2/§2.3; `cableTrace`(P2 완료, `frontend/src/features/trace/cableTrace.ts`).

## 배경 데이터 형태 (현재)
- **FiberPath**(raw row): `{ id, ofdAId, ofdBId, portCount }`.
- **fiber 케이블**: `{ id, sourceAssetId, targetAssetId, cableType:'FIBER', fiberPathId, fiberPortNumber }` — OFD(ofdAId|ofdBId) ↔ 로컬 설비.
- **FiberCore**: `{ id, fiberPathId, coreNumber, purpose, circuitText, spliceType, usageOverride }`.
- 변환 결과는 P2 `cableTrace` 의 `TraceAsset{ id, connectionKind }` / `TraceCable{ id, cableType, sourceAssetId, targetAssetId, sourceRole, targetRole, number }`.

### 변환 규칙
FiberPath `fp` 당:
- 슬롯 가상자산 2개: `slotId(fp.id,'A')='ofdslot:'+fp.id+':A'`, `:B`. 둘 다 `connectionKind:'conduit'`.
- OPGW 가상케이블: `{ id:'opgw:'+fp.id, cableType:'FIBER', sourceAssetId:slotA, targetAssetId:slotB, sourceRole:'IN', targetRole:'IN' }`.
fiber 케이블 `c`(fiberPathId=fp.id) 당:
- OFD-end = `c.sourceAssetId===fp.ofdAId || c.targetAssetId===fp.ofdAId` 이면 slotA, ofdBId 쪽이면 slotB. 설비-end = 반대쪽 assetId.
- OUT 가상케이블: `{ id:'out:'+c.id, cableType:'FIBER', sourceAssetId:slot, targetAssetId:설비, sourceRole:'OUT', targetRole:null, number:c.fiberPortNumber }`.
- 설비 자산은 `connectionKind:null`(passive)로 결과 assets 에 포함(중복 제거).
(슬롯도 결과 assets 에. FiberCore 는 이 어댑터 단계선 트레이스에 불필요 — 선번장 메타용이라 P3b 그리드에서. 시그니처엔 받되 미사용 가능.)

    expect(g.assets.map((a) => a.id)).toEqual(expect.arrayContaining(['ofdslot:fp1:A', 'ofdslot:fp1:B']));

### Step 3: 구현 — Create `frontend/src/features/trace/fiberAdapter.ts`:
```ts
import type { TraceAsset, TraceCable } from './cableTrace';

interface FiberPathRow { id: string; ofdAId: string; ofdBId: string; portCount?: number }
interface FiberCableRow {
  id: string; cableType?: string | null;
  sourceAssetId?: string | null; targetAssetId?: string | null;
  fiberPathId?: string | null; fiberPortNumber?: number | null;
}
interface FiberCoreRow { id: string; fiberPathId: string; coreNumber: number }

export interface UnifiedGraph { assets: TraceAsset[]; cables: TraceCable[] }

const slotId = (fpId: string, side: 'A' | 'B') => `ofdslot:${fpId}:${side}`;

/**
 * 기존 광 데이터(FiberPath/케이블)를 통일 그래프(conduit 슬롯 + OPGW + 번호 OUT 케이블)로
 * *읽기 변환*한다. cableTrace 가 이 그래프로 광경로를 낸다(파괴적 마이그레이션 전 검증·되돌리기 가능).
 * 라이브 데이터는 안 바꾼다 — 이건 메모리 변환기다.
 */
export function fiberToUnifiedGraph(
  fiberPaths: FiberPathRow[],
  fiberCables: FiberCableRow[],
  _fiberCores: FiberCoreRow[],
): UnifiedGraph {
  const assetById = new Map<string, TraceAsset>();
  const cables: TraceCable[] = [];
  const addAsset = (id: string, kind: TraceAsset['connectionKind']) => {
    if (!assetById.has(id)) assetById.set(id, { id, connectionKind: kind });
  };

  const byPath = new Map<string, FiberPathRow>();
  for (const fp of fiberPaths) {
    byPath.set(fp.id, fp);
    const a = slotId(fp.id, 'A'), b = slotId(fp.id, 'B');
    addAsset(a, 'conduit');
    addAsset(b, 'conduit');
    cables.push({ id: `opgw:${fp.id}`, cableType: 'FIBER', sourceAssetId: a, targetAssetId: b, sourceRole: 'IN', targetRole: 'IN' });
  }

  for (const c of fiberCables) {
    if (!c.fiberPathId) continue;
    const fp = byPath.get(c.fiberPathId);
    if (!fp) continue;
    // OFD-end 판정: 케이블의 두 끝 중 fp.ofdAId/ofdBId 인 쪽이 슬롯, 반대쪽이 설비.
    let slot: string | null = null;
    let equip: string | null = null;
    if (c.sourceAssetId === fp.ofdAId) { slot = slotId(fp.id, 'A'); equip = c.targetAssetId ?? null; }
    else if (c.targetAssetId === fp.ofdAId) { slot = slotId(fp.id, 'A'); equip = c.sourceAssetId ?? null; }
    else if (c.sourceAssetId === fp.ofdBId) { slot = slotId(fp.id, 'B'); equip = c.targetAssetId ?? null; }
    else if (c.targetAssetId === fp.ofdBId) { slot = slotId(fp.id, 'B'); equip = c.sourceAssetId ?? null; }
    if (!slot || !equip) continue;
    addAsset(equip, null);
    cables.push({
      id: `out:${c.id}`, cableType: 'FIBER',
      sourceAssetId: slot, targetAssetId: equip,
      sourceRole: 'OUT', targetRole: null,
      number: c.fiberPortNumber ?? null,
    });
  }

  return { assets: [...assetById.values()], cables };
}
```
